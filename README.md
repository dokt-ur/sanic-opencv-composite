# sanic-opencv-composite

Demo app

#sanic #opencv #aiohttp

### Highly recommended to create virtual environment first!

```
python3 -m venv tmp-env
source tmp-env/bin/active
```

# Run the app

`./bin/run`

-> app will be running on "http://localhost:8000"

# Linting

`./bin/lint`

# Tests

`./bin/test`
